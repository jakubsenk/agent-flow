# User Input (Verbatim)

Potřebuju co nejdříve dodat scaffold-to-deployment workflow pro ceos-agents plugin. Workflow:
1. SCAFFOLD MODE: V issue trackeru vytvořit nový projekt, přidat první epic, pak spustit scaffold pipeline
2. FEATURE LOOP: Po scaffoldu přidávat features do trackeru a spouštět implement-feature na každý
3. FORGE INTEGRATION: Volitelně použít filip-superpowers forge pro rozložení epicu/feature na tracker karty, pak spustit feature mode na každou kartu
4. LOCAL DEPLOYMENT: Cíl je mít appku deploynutou lokálně s DB, FE, BE
5. FUTURE: Integrovat celou věc na samostatný stroj

Uživatel říká, že to nemá plně promyšlené — je to exploratorní design úkol.
