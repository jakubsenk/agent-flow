# Phase 0 — User Input (Verbatim)

Audit definic agentů a skillu implement-feature v ceos-agents pluginu. Zkontrolovat, zda jsou agenti správně nadefinovaní pro feature workflow (nejen bug-fix) — role, popisy, process steps, constraints. Konkrétní obavy: fixer agent říká 'specializing in surgical bug fixes' ale řeší i feature development. Projít celý implement-feature skill a všechny agenty které dispatčuje, posoudit kvalitu definic, konzistenci a vhodnost pro feature pipeline.
