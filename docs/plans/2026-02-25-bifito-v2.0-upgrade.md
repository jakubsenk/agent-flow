# BIFITO — upgrade na CLAUDE-agents v2.0.0

## Předpoklady

- CLAUDE-agents v2.0.0 pushnutý na remote
- Přístup k BIFITO repozitáři (branch `development`)

## Postup

### 1. Aktualizace pluginu

```bash
cd ~/.claude/plugins/marketplaces/CLAUDE-agents && git fetch origin && git pull origin main
rm -rf ~/.claude/plugins/cache/CLAUDE-agents/
```

Restartuj Claude Code session.

### 2. Úpravy BIFITO CLAUDE.md

#### 2a. Přidat `Type` key do Issue Tracker

Přidej řádek `| Type | youtrack |` do tabulky Issue Tracker:

```markdown
### Issue Tracker
| Klíč | Hodnota |
|------|---------|
| Type | youtrack |          ← NOVÝ ŘÁDEK
| Instance | `...` |
| Project | `BIFITO` |
| ...  | ... |
```

**Poznámka:** Default je `youtrack`, takže pipeline funguje i bez tohoto řádku. Ale explicitní hodnota je lepší pro jasnost a pro případ budoucího přechodu na jiný tracker.

#### 2b. Ověřit PR Description Template

v2.0.0 očekává `### PR Description Template` jako **separátní subsekci** (ne uvnitř PR Rules). Ověř, že v BIFITO CLAUDE.md je:

```markdown
### PR Rules
| Klíč | Hodnota |
|------|---------|
| Labels | `ForReview` |

### PR Description Template

## Summary
{summary}
...
```

Pokud je template uvnitř PR Rules tabulky → přesuň do vlastní subsekce.

### 3. Validace

```
/CLAUDE-agents:check-setup
```

Očekávaný výsledek: všechny povinné sekce [OK], MCP servery [OK], konektivita [OK].

### 4. Volitelné rozšíření (nice to have)

| Sekce | Co přidá | Doporučení |
|-------|----------|------------|
| Retry Limits | Konfigurovatelné limity (fixer 5x, test 3x, build 3x) | Přidat — defaults jsou rozumné, ale explicitní je lepší |
| Hooks | Pre/post hooks na pipeline kroky | Zvážit — např. `Pre-fix: dotnet build` jako guard |
| Notifications | Webhook notifikace | Zvážit pro Slack/Teams integraci |
| Error Handling | Max blocked per run | Přidat pokud se spouští `/fix-bugs` s vyšším N |

Pro generování kompletního Automation Config bloku lze použít nový command:

```
/CLAUDE-agents:onboard
```

### 5. Smoke test

Po úpravách spusť dry-run na existujícím bugu:

```
/CLAUDE-agents:fix-ticket BIFITO-XXX --dry-run
```

Ověří: triage + code-analyst fungují, výstupem je dry-run report bez side effects.
